# blueskynet: Analyze and Visualize Networks on Bluesky Social

Provides comprehensive tools for collecting, analyzing, and visualizing
social networks from Bluesky Social <https://bsky.app>. Enables users to
authenticate with the Bluesky API, collect follower networks, perform
community detection, calculate network metrics, and create interactive
3D visualizations. Includes functions for network expansion algorithms,
text analysis of user profiles, and follow/unfollow management. Designed
for social network researchers, data scientists, and anyone interested
in understanding social structures on Bluesky Social.
